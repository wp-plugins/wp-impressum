Load your Imprint (Impressum) from a central space you own.

== Description ==

Every site need a imprint. If you own more then one page you get problem with change your imprint for all site.
With wp_impressum you will never forget update a imprint for all of your sites.
Put your imprint to a local Space like dropbox and load the file for all your imprint pages.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp_impressum.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Put the Path to the imprint in the config
4. Set the marker to the imprint page
5. Finish! get save of updates your imprint.
